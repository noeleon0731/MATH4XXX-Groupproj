# Cancer Classification using KNN and SVM

This is a data science project focused on evaluating the effectiveness of two supervised machine learning algorithms (K-Nearest Neighbors and Support Vector Machines) in classifying tumors as either malignant or benign using the Wisconsin Breast Cancer Dataset.

## Key Implementations:
- Data normalization and feature scaling
- Hyperparameter optimization 
- Model evaluation via confusion matrices and accuracy scoring
